## 🆘 Troubleshooting Quick Fixes

| Problem | Solution |
|---------|----------|
| `git not found` | Install from https://git-scm.com/ |
| `fatal: not a git repository` | Run `git init` first |
| `authentication failed` | Use Personal Access Token instead of password |
| `permission denied` | Check SSH keys or use HTTPS |
| `everything up to date` | Your files are already pushed! |
| `Repository not found` | Check username/repo name in git remote |
